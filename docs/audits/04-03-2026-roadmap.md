## Architectural (Long-term)

- [ ] Replace `SharedPreferences` with Hive or Isar for `ReachData` persistence — SharedPreferences is not designed for large serialized objects; no queryability, no schema migration path *(audit §2.2)*
- [ ] Add request cancellation — cancel in-flight HTTP calls when a provider or widget is disposed; currently dangling requests update state on dead providers *(audit §1.4)*
- [ ] Migrate to Clean Architecture — introduce domain entities, repository interfaces, and use case classes per feature; lowest-scoring category (14/90) and the one that unlocks testability of all data sources *(audit §8)*
